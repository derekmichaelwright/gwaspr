# theme_gwaspr.

ggplot theme.

## Usage

``` r
theme_gwaspr(
  x,
  bgFill = "white",
  lineColor = "grey90",
  linesize = 0.75,
  stripFill = "white",
  ...
)
```

## Arguments

- x:

  ggplot

- bgFill:

  background fill color

- lineColor:

  color of axis lines

- linesize:

  border line size

- stripFill:

  strip background color

## Value

ggplot with gwaspr theme
