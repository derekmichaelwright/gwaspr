gwaspr R Package
================

`gwaspr`: an `R` package for plotting GWAS results from the `GAPIT`
package

![](man/figures/logo_gwaspr.png)

------------------------------------------------------------------------

# Instal gwaspr

``` r
devtools::install_github("derekmichaelwright/gwaspr")
```

``` r
library(gwaspr)
```
