# convert_IUPAC

Converts hapmap formatted genotype files to and from IUPAC codes.

## Usage

``` r
convert_IUPAC(xg, toIUPAC = T)
```

## Arguments

- toIUPAC:

  Logical, convert to IUPAC (TRUE) or from IUPAC (FALSE).

- xG:

  GWAS genotype object. Note: needs to be in hapmap format.

## Value

A LD Heatmap plot.
