# gg_plotly

Creates an interactive plotly plot. Note: This function will require
`plotly` and `htmlwidgets` packages.

## Usage

``` r
gg_plotly(mp, filename = "my_gg_plotly.html", showlegend = T)
```

## Arguments

- mp:

  ggplot object.

- filename:

  Filename for plot.

- showlegend:

  Logical, should the legend be displayed.

## Value

A plotly summary GWAS plot.
